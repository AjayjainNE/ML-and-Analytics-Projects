## 📁 Projects

### 🏥 Heart Disease Prediction — End-to-End ML Pipeline

Real-world heart disease dataset processed through a full pipeline — raw → cleaned → normalised → standardised — then modelled across five algorithms for comparison.

**Data:** `HeartDisease2026.csv` → `heart_disease_cleaned.csv` → `heart_disease_normalized.csv` → `heart_disease_standardized.csv`

| Notebook | Algorithm | Key Skills |
|---|---|---|
| **[decision_tree.ipynb](./decision_tree.ipynb)** | Decision Tree | Feature splitting, pruning, classification, evaluation |
| **[random_forest.ipynb](./random_forest.ipynb)** | Random Forest | Ensemble learning, feature importance, overfitting control |
| **[lasso_regression.ipynb](./lasso_regression.ipynb)** | Lasso Regression | L1 regularisation, feature selection, regression metrics |
| **[kmean_cluster.ipynb](./kmean_cluster.ipynb)** | K-Means Clustering | Unsupervised learning, cluster analysis, segmentation |
| **[NN.ipynb](./NN.ipynb)** | Neural Network | Feedforward network, backpropagation, binary classification |
